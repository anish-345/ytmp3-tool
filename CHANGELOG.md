# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-28

### Added
- 🎵 `ytmp3` command for YouTube to MP3 conversion
- 📱 Android intent sharing support via `termux-url-opener`
- 🔄 `ytmp3-auto` for clipboard-based conversion
- 📋 `ytmp3-widget` to show recent MP3s
- 🚀 `install.sh` for easy installation
- 📚 Comprehensive README documentation

### Features
- Real MP3 encoding (uses ffmpeg, not just renaming)
- Highest quality VBR 0 encoding
- Automatic URL detection from shared intents
- Clipboard support for easy conversion
- Organizer-ready output directory
